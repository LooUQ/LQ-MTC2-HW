# LooUQ MTC.2 Hardware

Welcome to the hardware and physical parts repository for the LooUQ MTC.2 Ecosystem. Here you will find resources to aid in your implementation of MTC.2 devices in your design/product. Many parts of the LooUQ MTC.2 ecosystem are available as open-source hardware and there design files are available here. There are also drawings for components that can be 3D printed. 

Software developement tooling is found in the LQ-MTC2-SW repository, which is also found on LooUQ's GitHub site.
